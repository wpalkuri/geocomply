Description
This project is designed to read basefile and create multiple files based on file extension.
Input to the project is basefile.txt

output:
It automatically creates new files as cfile.txt, cppfile.txt, csfile.txt

steps to execute
open the project in eclipse
run as java application

Pre req:
copy the project in this path 
C:\SeleniumWorkspace\geocomply

(or) create your own folder and update path of your choise in main()
